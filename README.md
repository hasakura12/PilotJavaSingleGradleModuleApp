# PilotJavaSingleGradleModuleApp

## How to build the project
`./gradlew build`

This will compile the project and run tests

## How to run only tests
`gradle test`


## How to run the application
just execute

`./gradlew run`

or

`cd build/lib`

then,

`java -jar PilotJavaSingleGradleModuleApp-1.0.jar `
